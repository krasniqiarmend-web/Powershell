# Powershell

Powershell Exchange Administration



\# Exchange PowerShell Toolkit



Collection of PowerShell scripts used for Microsoft Exchange administration and troubleshooting.



Features:



\- Mail flow diagnostics

\- Message trace investigation

\- Exchange service health checks

\- Autodiscover troubleshooting

\- Mailbox provisioning

\- Hybrid mailbox migration preparation



Environment:



\- Exchange Online

\- Exchange Server

\- Hybrid Exchange



Example:



Test-MailFlow -Sender admin@company.com -Recipient user@company.com
